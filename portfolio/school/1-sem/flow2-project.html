Flow 2 project

Scope

Using the tools HTML5 and CSS3 we learned in flow 2, we had to implement an online version of our portfolio. Besides the website itself, we had to upload a repository to GitHub with appropriate naming of files and folders, and our code must be validated.

Full project description

Initial idea

My very first thought was that it would be cool to implement a logo on my website, so I planned on venturing into Illustrator to create it. I really liked the font I used for my prototype, and decided to continue with that. I also wanted to create dropdown in my menu, to show that I was able to make it work in my code. My colour scheme was the same as my prototype, with some slight alterations, but I scrapped the header with the background photo, my picture and my name.

Challenges and process

Contrary to when I made my prototype, I knew that I now wanted my front page to feature information about the skills I already have learned as well as what I am going to learn during the course of my studies, to keep it professional. My portfolio part got sectioned into "professional work" and "school projects", although there would only be content in the school section, just to show that there's room for the site to grow. I also sectioned my about further than with my prototype, adding my resume but splitting it between education and work experience. Lastly, based on the feedback for my prototype, I made a contact form. For my elements in general, I felt like borders looked very heavy and clumsy, so I used shadows instead, which also added some depth to the site overall.

Product

View website version 1.1 in a new tab

Feedback

I got very positive feedback on my site, especially my logo which wasn't even a requirement of the project. My professors specifically mentioned that they liked that I had included "scope" and "ideas" sections to my project pages, but said that it would be a good idea to make my menu logo into a link that referenced back to the front page, which I already thought about when I presented the site, because I kept wanting to click it by reflex myself.
